# Asset Credits

This project mostly uses project-owned assets made specifically for the submission.

## Project visual art
- `assets/tiles/`, `assets/blockpack/`, `assets/decor/`, and `assets/food/`: project-owned tile, block, prop, tree, and food art. These are the current runtime world assets and were AI-generated/edited for this game rather than pulled from the old asset packs.
- `assets/characters/player_new.png` and `assets/characters/bob_new.png`: project-owned AI-generated character art for the player and B.O.B.
- `assets/tools/tool_strip.png`: project-owned tool icon strip used by the HUD and inventory-style crafting menu.
- `assets/ui/bob_attack_start_screen.png` and `assets/ui/*_ui16.png`: project-owned start-screen and HUD UI art.

## Font
- `assets/fonts/PressStart2P-Regular.ttf`
- Source: https://fonts.google.com/specimen/Press+Start+2P

## Procedural gameplay SFX (`assets/audio/sfx/`)
- Short WAV files generated locally (tones/noise) for mine, UI, craft, and B.O.B. feedback.
- Treated as original project audio for this submission.

## Removed prototype/reference assets
- Earlier prototype/reference folders such as Kenney, KayKit, Tiny16, roguelike sprites, local study notes, and course resource zips are not required by the current game and are not credited as active runtime assets.
